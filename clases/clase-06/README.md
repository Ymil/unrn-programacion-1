# Clase 6

## Temas de la clase

- Repaso de tipos de datos en Python.
- Recorrido de listas con `for`.
- Acumuladores, contadores y promedio.
- Busqueda de maximos y minimos sin usar `max` ni `min`.
- Filtros sobre listas usando condiciones.
- Primer y ultimo elemento de una lista.
- Repaso de funciones, parametros y valores de retorno.
- Entrada de datos con `input` y corte de repeticion con `while`.
